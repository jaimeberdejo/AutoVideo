class VoiceMode(str, Enum):
    elevenlabs = "elevenlabs"
    record = "record"

    hybrid = "hybrid"
    manual = "manual"

class RunConfig(BaseSettings):
    bullets: Path

    level: int = Field(default=4, ge=1, le=4)

    dry_run: bool = False
    burn_subs: bool = False
    verbose: bool = False
    workdir: Path = Path("workdir")

    language: str = "es"

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user filesystem → RunConfig | `--bullets` / `--context` paths and `config.yaml` are user-controlled input crossing into the program |
| config.yaml → process | YAML content is parsed into settings |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | YAML config parsing (RunConfig YamlConfigSettingsSource) | mitigate | Config is parsed exclusively via pydantic-settings YamlConfigSettingsSource which uses yaml.safe_load under the hood; no yaml.load anywhere |
| T-01-02 | Information Disclosure | config.yaml committed to git with secrets | mitigate | config.yaml contains only non-secret defaults (wpm, voice_id, level, language); acceptance criterion greps that no api_key appears; API keys live in .env (gitignored) |
| T-01-03 | Tampering | Invalid duration/level reaching pipeline | mitigate | RunConfig Field(gt=0) on duration and Field(ge=1, le=4) on level reject out-of-range values at construction (covered by test_models.py) |
| T-01-04 | Tampering | Partial/corrupt checkpoint on interrupt | mitigate | Atomic write tmp→os.replace; done marker written only after replace succeeds; interruption test asserts no partial file (test_workdir.py) |
</threat_model>
